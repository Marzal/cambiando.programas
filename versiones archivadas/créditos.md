# Créditos y atribuciones

## Trabajo y sitio Web original

"[switching.social][fuente]" fue una publicación anonimia bajo la licencia [CC BY-SA 4.0][lisesp].

[lisesp]: https://web.archive.org/web/20190802083143/https://creativecommons.org/licenses/by-sa/4.0/deed.es

> El proyecto Switching social fue abandonado misteriosamente en Septiembre 18 2019, por lo que se comenzó un nuevo proyecto bajo el nombre de [switching.software][fuente2].

## Switching Software

"[switching.software][fuente2]" de [Tobias][tobi] es un trabajo derribado de "[switching.social][fuente]", ambos bajo la licencia [CC BY-SA 4.0][lisesp].

> [Tobias][tobi] es el escritor de la versión Alemana de [switching.social][fuente]. Después de la desaparición del trabajo original Tobias comenzó Switching Software para mantener el proyecto a flote. Ahora el mantiene ambas versiones, en Inglés y en Alemán.

## Repositorio "switching.software" de Rosnovsky

El repositorio "[switching.software][fuente-repo]" de [Artem Rosnovsky][rosni] es un trabajo derribado de "[switching.software][fuente2]", ambos bajo la licencia [CC BY-SA 4.0][lisesp].

[fuente-repo]: https://github.com/rosnovsky/switching.software

> El usuario de GitHub [Rosnovsky][rosni-git] usó [web.archive.org](https://web.archive.org/web/20190915101437/https://switching.social/) y google cache para archivar el proyecto [switching.software][fuente2] mantenido por [Tobias][tobi]. Esta traducción se basa grandemente en esta versión archivada.
